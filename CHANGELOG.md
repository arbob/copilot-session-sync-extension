# Changelog

All notable changes to the "Copilot Session Sync" extension will be documented in this file.

## [0.1.0] - 2026-02-17

### Added

- Initial release
- Cross-device Copilot chat session sync via private GitHub repository
- AES-256-GCM encryption with user passphrase
- Automatic sync on startup and periodic intervals
- Per-workspace sync exclusion
- Conflict resolution with automatic backups
- Status bar indicator showing sync state
- Commands: Sync Now, Set Passphrase, Toggle Workspace, View Status, Reset Sync
- Configurable sync interval, max session age, and repo name
